# AI Setup Guide - FurnitureAI Professional

## Overview

FurnitureAI integrates AI capabilities for intelligent furniture generation.

## Supported Providers

### Local Providers (Free)
- **LM Studio**: https://lmstudio.ai
- **Ollama**: https://ollama.ai

### Cloud Providers (Paid)
- **OpenAI**: https://platform.openai.com
- **Anthropic**: https://console.anthropic.com

### Custom Server
See: [NPU Server Setup](./npu_server_setup.md)

## Configuration

1. FurnitureAI → Settings → Configure AI
2. Select provider
3. Configure settings
4. Test connection
